--- layout: post title: 'Radiate: A simple push server for SocketIO (alpha)' date: '2011-10-03T11:45:00.001+08:00' author: kagesenshi tags: - Python - Open Source - Hacks - Javascript - HTML5 modified_time: '2011-10-03T11:48:10.092+08:00' blogger_id: tag:blogger.com,1999:blog-28371165.post-5607457449277804586 blogger_orig_url: https://blog.kagesenshi.org/2011/10/radiate-simple-push-server-for-socketio.html --- Based on my previous post on WebSocket server using Tornado, I've hacked the code more to be a SocketIO server instead of just WebSocket, so that it can be used on many browsers until everyone support WebSocket. The SocketIO connection is handled by TornadIO, a neat SocketIO library for Tornado.

I've uploaded the code here in github: https://github.com/kagesenshi/radiate

I've packaged Radiate together with a buildout.cfg, so you can deploy it easily by just using these few lines of commands
git clone https://kagesenshi@github.com/kagesenshi/radiate.git radiate
cd radiate
python bootstrap.py
./bin/buildout
To start the daemon, just execute:
./bin/radiate-admin fg
The readme file on how to deploy and use Radiate is here: https://github.com/kagesenshi/radiate/blob/master/README.txt.
