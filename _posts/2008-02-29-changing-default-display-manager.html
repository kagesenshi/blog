The default display manager in Fedora GNOME is GDM. But sometimes, you might want to switch to a different display manager such as KDM or XDM. One way of doing it is through removing the other DMs and only leave one DM intact. But thats ugly!.

Theres an easy way to do it, which is through /etc/sysconfig/desktop. Create it if its not yet exist and add this line.

DISPLAYMANAGER=KDE

Valid values are:

DISPLAYMANAGER=GNOME
DISPLAYMANAGER=KDE
DISPLAYMANAGER=WDM
DISPLAYMANAGER=XDM
